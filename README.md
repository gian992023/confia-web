# confia-web

Landing pública de **ConfIA** — IA y automatizaciones para pymes de Yopal, Casanare (Colombia).

Sitio estático (HTML + Tailwind por CDN + JS). Se publica con **GitHub Pages**.

## Desarrollo local

```bash
python -m http.server 8080
# abrir http://localhost:8080
```

## Editar datos de contacto

Edita el bloque `CONFIG` al inicio de `main.js` (WhatsApp, correo, redes).

## Publicación

GitHub Pages → *Settings → Pages → Deploy from a branch → `main` / `/ (root)`*.
URL: https://gian992023.github.io/confia-web/
